# Bioenergetic Stability Index (ISB): A Computational Framework

## Overview
This repository contains the complete custom Python codebase utilized for the numerical integration, topological bifurcation analysis, and *in silico* Pan-Ancestry simulation detailed in the manuscript: **"Mechanistic Reconstruction of Integrative Psychopathology: A Theoretical Framework and Pan-Ancestry In Silico Validation of the Bioenergetic Stability Index (ISB)"**. 

This repository is provided to ensure 100% computational reproducibility of the mathematical models and phase-space projections discussed in the study.

## Mathematical Architecture & Zero-Rounding Protocol
The system dynamics are modeled via thermodynamic Coupled Ordinary Differential Equations (ODEs) to capture the non-linear trajectories of astrocytic ATP ($A$) and extracellular Glutamate ($G$). 

To maintain strict methodological parsimony and avoid empirical curve-fitting, the governing equations strictly utilize exact fractional mathematics for parameterization derived from fundamental constraints (e.g., $k_v = 7/200$, $\beta = 1/10$). No intermediate decimal rounding is applied during the derivation of the Jacobian matrix stability criteria.

## Repository Structure
* `01_ISB_Metadata_Extraction.py`: The data mining module responsible for programmatic retrieval of genomic and metabolic summary statistics from the NCBI PMC database.
* `02_ISB_ODE_Simulation.py`: The primary numerical integration engine containing the thermodynamic ODE system, limitation functions ($\Phi_{vaso}$ and $\Phi_{bohr}$), and Pan-Ancestry simulation execution.
* `ISB_PanAncestry_Simulation_Results.csv`: The output dataset containing the terminal phase-space coordinates for the modeled cohorts.
* `requirements.txt`: List of required Python libraries (NumPy, SciPy, Pandas, Biopython).

## Installation & Execution
To reproduce the simulated cohorts and identify the Saddle-Node Bifurcation boundaries:
1. Ensure Python 3.8+ is installed.
2. Install the required dependencies: `pip install -r requirements.txt`
3. Execute the simulation script: `python ISB_ODE_Integration.py`

## Explicit Assumptions & Boundary Conditions
In adherence to computational rigor, the following theoretical boundaries of this simulation must be explicitly noted:
1. **Domain Separation:** This codebase represents an *in silico* mathematical projection. The simulated state variables (ATP < 0.5 mM as a topological collapse boundary) are theoretical derivations and do not represent definitive *in vivo* clinical diagnostics.
2. **Spatial Homogeneity:** The current ODE architecture operates under the assumption of uniform spatial distribution and does not account for the non-linear complexity of asymmetric spatial diffusion within specific limbic sub-regions.
3. **Absence of Ceiling Constraints:** The model fundamentally captures depletion dynamics. It currently lacks a thermodynamic capping function (negative feedback inhibition) for maximal ATP reserves under prolonged integration limits.

## Future Directions
The outputs generated by these scripts are intended exclusively for hypothesis generation. Translating these temporal bioenergetic perturbations into macroscopic clinical phenotypes theoretically requires prospective *in vivo* validation using multimodal instrumentation (e.g., 1H-MRS) and integration with complex spatial network topologies.
